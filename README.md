## ⚗️ Installation

Clone the project

```bash
  git clone https://github.com/rupam-seal/drone-website.git
```

<br>

Go to the project directory

```bash
  cd drone-website

```

<br>

Install Dependencies

```bash
  npm install

```

<br>

Run server

```bash
  npm start
```
